# Petly Mobile

Primera version de la app movil Petly con Expo.

## Pantallas iniciales

- Login demo.
- Feed de comunidad.
- Mascotas.
- Crear publicacion.
- Perfil.

## Ejecutar

```bash
cd apps/mobile
npm install
npm run start
```

## Siguiente integracion

Conectar estas pantallas con Supabase usando las mismas entidades del MVP web:
